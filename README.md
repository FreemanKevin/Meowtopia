## Meowtopia

### Quick Start
```shell
# clone the project
git clone https://github.com/FreemanKevin/Meowtopia.git
cd Meowtopia

# run the project
mkdocs build
mkdocs serve

# update the themes
vim mkdocs.yml

# deploy the project
mkdocs build
mkdocs gh-deploy

# commit the code
git pull
git add .
git commit  -m "update files."
git push
```
