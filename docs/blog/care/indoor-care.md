# 室内猫咪环境布置

## 基础设施

### 猫砂盆
- 数量：猫咪数量+1
- 位置：安静、通风处
- 清理：每天至少1-2次
- 选择：
  1. 开放式/封闭式
  2. 适合大小
  3. 易于清洁

### 食水区
- 食盆与水盆分开
- 避免靠近猫砂盆
- 定期清洗消毒
- 建议使用活水机

## 休息区域

### 睡眠空间
- 提供多个选择
- 高低搭配布置
- 柔软舒适
- 避免穿堂风

### 攀爬设施
- 猫爬架
- 窗台垫
- 墙面跳台
- 确保安全稳固

## 娱乐设施

### 玩具配置
- 互动型玩具
- 自嗨型玩具
- 智力玩具
- 定期更换保持新鲜感

### 磨爪设施
- 猫抓板
- 猫抓柱
- 合理布置位置
- 及时更换
